## AI

## Raisonnement : Induction, Déduction, Abduction


Epistémologie



L'induction est historiquement le nom utilisé pour signifier un genre de raisonnement qui se propose de chercher des lois générales à partir de l'observation de faits particuliers, sur une base probabiliste.

Historiquement :

- IA symbolique (années 70–90) → approche plutôt déductive
règles logiques explicites
- IA moderne (machine learning) → approche inductive
apprentissage depuis les données
